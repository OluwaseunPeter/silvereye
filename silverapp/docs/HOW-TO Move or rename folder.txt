Steps to setup OGRE3D with Qt when moving/renaming the source code folder:

Rename the folder and the OldName.pro file.
Remove the OldName.pro.user

Double click the NewName.pro file,
Only check the option "Desktop Qt 5.7.0 MSVC2013 32bit".
"Configure Project"

Open NewName.pro
Update TARGET (TARGET = NewName)

Setup the OGRE3D specific compilation actions with QtCreator
Source: http://www.ogre3d.org/tikiwiki/tiki-index.php?page=Setting+Up+An+Application+-+QtCreator
(A snapshot of this page has been provided to prevent link rot)

Guide should be followed from "Build Settings" until, but not including, "Tutorial framework".
